AL-HALA CANDIES — BRAND ASSETS · V1.0 (July 2026)
====================================================

The mark: a solid deep-green tile crossed by a gilded square —
together forming an eight-point star, holding the name in both scripts.

COLORS
  Hala Green (core)   #1F5A3D
  Deep Green (tile)   #0E1F17   (dark mode tile: #17402E)
  Saffron Gold        #C98B1F   (dark mode ring: #E9C87E)
  Cream               #FAF7F0
  Cocoa Ink           #221B14

FONTS (Google Fonts, free)   · v1.1, amended July 2026
  Cormorant Garamond — display / wordmark   (weight 300 at display sizes)
  Jost — body & captions
  Amiri — Arabic script (الحلا / ح)

  SUPERSEDED in v1.1:
    Marcellus (display) and Albert Sans (body).
    Marcellus is a Roman capital face with almost no stroke contrast, so it never
    reads as dramatic however large it is set. The brand needs thick-thin to carry
    "premium" at display sizes. Cormorant Garamond is a Garamond drawn razor-thin;
    its contrast does that work. Set it at 300 above 3rem — at 400+ the hairlines
    thicken and the drama is lost.

    The SVG/PNG assets in this kit still have Marcellus type set into the wordmark
    and have NOT been re-cut. Re-set them before any print production.

RULES
  Clear space = height of the ح glyph on all sides.
  Minimum sizes: mark 24px / 10mm; horizontal lockup 120px / 30mm.
  Never recolor, stretch, add effects, or place on busy imagery.

NOTE
  These marks are type-set studies (live Amiri/Marcellus type).
  Before large-scale print production, have a lettering artist
  refine the ح and الحلا into custom-drawn semi-Kufic letterforms.
